### 🧐 关于我

长期折腾 CLI 工具和嵌入式开发，偶尔跨到 audio / DL 方向探索。习惯把真正用过、踩过坑、且值得复用的东西做成开源仓库持续迭代下去。

- 🔧 **嵌入式 + DL 落地**：`power-line-detection` (YOLOv5s + RK3588 NPU 端侧部署)
- 📖 **学习路径**：`from-zero-to-dl` — 从 matplotlib 到 deep learning 的中文学习记录
- 💬 **常聊**：Rust、STM32、PyTorch、Claude / Codex 工作流、电子竞赛
